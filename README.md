Description
-----------------------------------

This project includes a set of console games, where you can solve some problems.
For example:

    What number is missing in this progression?
    5 7 9 11 13 .. 17 19 21 23


Installation
-----------------------------------

    npm install brain-games-s280


Usage
-----------------------------------

    $ brain-even
    $ brain-calc
    $ brain-gcg
    $ brain-progression
    $ brain-balance
    $ brain-prime
